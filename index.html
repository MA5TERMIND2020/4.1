The encoding of pixel location on a web browser is very similar to the 4th quadrant of a cortesean plane.
The only difference is that the Y-values are inverted. The origin (0,0) is located in the top-left corner of
the window and as the X value increases, the location of the pixel being referenced moves to
the right. But as the Y value increases, the location of the pixel being referenced moves
down. This web page provides a good way to visualize this by allowing you to see the
coordinate point values displayed on the screen in real time
as an image is moved across the screen.
